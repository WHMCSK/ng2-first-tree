组件输入

输入 Type 描述
[settings] Object 表组件配置对象，属性描述如下
[source] Array 树数据，数组

树配置

属性 Type 默认 描述
menuDatas 数组 空数组 右键菜单的数据结构，menuDatas: [ {{left}}title: 'add', text: '新增'{{right}}, {{left}}title: 'edit', text: '修改'{{right}}, ],
showicon string `icon ion-filing` 子节点展开时的图标， showicon：`icon ion-filing`
hideicon string `icon ion-folder` 子节点隐藏时的图标， hideicon：`icon ion-folder`
noChildicon string `icon ion-document-text` 没有子节点时的图标， wenjicon：`icon ion-document-text`
filter object filter: {{left}} type: '', class: '', isShow: false, {{right}} 控制搜索框的显示隐藏
selectBgc object selectBgc: {{left}} open: true, bgc:'#00abff', lv1: false, {{right}} 控制选中背景色的启用，颜色，第一级是否启用

树数据

属性 Type 默认 描述
text string 必填 在界面上显示的文字，text:"学校"
treeheight number 必填 用来控制是第几级节点，treeheight:1
co Boolean 必填 用来控制是否展开子节点，co:true
children 数组 无默认，没子节点就为空 子节点数据结构，内部数据结构跟上述结构相同
数据内的属性可根据需要自行添加，所有属性在点击事件的event对象里都能获取到

组件输出事件

事件名 Type 描述
nodeClick function 节点单击事件，返回值：当前点击的节点数据
nodeDblClick function 节点双击事件，返回值：当前点击的节点数据
nodeMenuClick function 右键菜单上的单击事件，返回值：右键点击的节点数据，菜单的title
searchEvent function 搜索按钮单击事件，返回值：搜索框内的值
allFolded function 全部折叠事件，返回值：tree所有的数据
allExpansion function 全部展开事件，返回值：tree所有的数据
