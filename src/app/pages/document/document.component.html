组件输入

输入 Type 描述
[settings] Object 表组件配置对象，属性描述如下
[source] Array 树数据，数组

表格配置

属性 Type 默认 描述
menu 数组 没有默认，必填 右键菜单的数据结构，menu:[{{left}} text:"添加兄弟节点", clickFn: ()=>{{left}}{{right}} {{right}}]
nodeclick function 没有默认，必填 节点上的点击事件， nodeclick: ()=>{{left}}{{right}}
showicon string 没有默认，必填 子节点展开时的图标， showicon：`icon ion-filing`
hideicon string 没有默认，必填 子节点隐藏时的图标， hideicon：`icon ion-folder`
wenjicon string 没有默认，必填 没有子节点时的图标， wenjicon：`icon ion-document-text`
searchShow Boolean false 控制搜索框的显示隐藏， searchShow：true | false
selectBgc object 关闭选中背景色 控制选中背景色的启用，颜色，第一级是否启用， selectBgc: {{left}} open: true, bgc:'#00abff', lv1: false, {{right}}

表格数据

属性 Type 默认 描述
text string 必填 在界面上显示的文字，text:"学校"
treeheight number 必填 用来控制是第几级节点，treeheight:1
co Boolean 必填 用来控制是否展开子节点，co:true
children 数组 无默认，没子节点就为空 子节点数据结构，内部数据结构跟上述结构相同
数据内的属性可根据需要自行添加，所有属性在点击事件的event对象里都能获取到
